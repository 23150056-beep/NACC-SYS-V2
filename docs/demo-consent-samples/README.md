Files for demoing the pre-assessment **consent step**: uploading one of these as
the "Scanned signed form" flips the consent status to **Signed** automatically.

| File | Child (seeded demo record) | Signed by |
|---|---|---|

The `.pdf` demos the scanned-PDF path of the upload field; the `.jpg` is the
same page as an image, for the "photo of the signed paper" path.

All documents are **generated and fully fictional** — the children and
psychologists are NACC-V2 seeded demo records, the guardians and signatures are
invented, and every page is watermarked "SAMPLE - FICTIONAL DATA - FOR SYSTEM
DEMO ONLY". They are not real consent records and must not be presented as such.

Regenerate or restyle with PyMuPDF if the demo needs different names — never
replace these with a scan of a real signed consent, which would be confidential.
