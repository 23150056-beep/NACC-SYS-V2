# Demo samples — psychological reports

Files for demoing the **Upload Report** function on Results & Reports (the
psychologist's own report, uploaded in their own format).

| File | Child (seeded demo record) | Report |
|---|---|---|
| `sample-psychological-report-progress-miles-morales.pdf` | Miles D. Morales (C-0014 · Adoption) | Progress report, Sessions 1-3 · by Levi D Makalaya, RPsy |
| `sample-psychological-report-initial-angelica-torres.pdf` | Angelica Torres (C-0009 · Adoption) | Initial evaluation, intake · by Hannah Baker, RPsy |

Both have a real text layer, so after upload the app extracts the text and the
**AI summary** button works on them.

All content is **generated and fully fictional** — children and psychologists
are NACC-V2 seeded demo records, every finding is invented, and each page is
watermarked "SAMPLE - FICTIONAL DATA - FOR SYSTEM DEMO ONLY". Never replace
these with a real psychological report, which would be confidential.

With this folder, every upload function has demo files:

- **Upload Report** → this folder
- **Upload Case Referral** → `../demo-case-referral-samples/`
- **Consent scanned signed form** → `../demo-consent-samples/`
